The main window consists of two major sections. The upper part (just under the toolbar), displays all topic titles of a certain category. The lower part displays a topic. The topic that is displayed can either always match the selected topic (this means that you only have to highlight a topic title to load it), or it can be an other topic then the one that is highlighted - depending on the settings in the configuration screen. In the latter case you must either double-click the topic title or hit the Enter key to load a specific topic.


Save, print and copy:

To save or print a topic you can select the matching commands under the File menu. To copy the current topic to the clipboard, you must first select all text (hint: use Ctrl-A) and then copy the text (Ctrl-C). A topic is always saved to disk as plain text, this means that the special topic formatting as used in RBIL Viewer, is lost. When copying the topic to the clipboard however, the topic formatting remains. This means that when you paste the text into a program that can handle Rich Text, the text will look the same as it does in RBIL Viewer (otherwise it will be pasted as plain text). For printing you can either choose to print a topic as it appears in RBIL Viewer (with current font and topic formatting), or to override these settings and just print the text in a special font without topic formatting. You can change this setting in the configuration screen. You can also print a partial topic by selecting the text you want to print and choosing Selection as the print range in the print dialog. 


Changing categories:

To display only the topics of a certain category, you can select that category from the category combobox which is located on the toolbar. The default selection is All categories when a list has more than one category, and the single category when a list has only one. When you are in a specific category and you want to switch back to All categories then you can just click on the word Categories (located on the left side of the combobox). 


Finding topics and text:

To find a certain topic title or to find text in the current topic, you can use the Find option in the Search menu. Just specify the text you want to find and press Ok. To repeat the last search operation you can select Search again or use the key F3. Click hereQF52ZN for more information about finding topics and text.


Zoom topic:

To hide the topic titles and to show only the current topic, you can select Zoom topic from the Options menu. You can also use the keyboard shortcuts F5 to zoom and Esc or F5 to unzoom. With a special setting in the configuration screen you can specify to always zoom a topic when the topic title is double clicked. (Note: this option is Off by default).
