Installation:

RBIL Viewer has no special installation procedure, just place the program files (Rbilview.exe and Rbilview.hlp) in any folder of your choice and make your own desktop shortcut if you like. 


Setting up the program:

The Open List menu is designed to display all list files of a certain directory; this makes it easy to switch between lists. You can find this menu under the File menu and on the toolbar (click on right section of the Open List button). When you run RBIL Viewer for the first time then the Open List menu will probably only display Open. To display all lists of The Interrupt List in the menu you must first specify the location of The Interrupt List in the configuration screen. The configuration screen can be found under the Options menu (Configurationâ€¦). You can now change the Location of list files. Click Ok to accept the changes and to close the configuration screen. The List menu should now display all lists of The Interrupt List. (see FAQ'sFAQS if Interrup.lst and Ports.lst do not show up in the menu).


Loading lists for the first time:

When you open a list for the first time (either by selecting it directly from the Open List menu or by using Other or Open), RBIL Viewer will automatically create an indexfile for that list. RBIL Viewer uses this indexfile so it can quickly load all topic titles the next time you open that list. Although you can disable the usage of indexfiles (in the configuration screen), it is best left enabled whenever possible since it only speeds up the loading of a list. Click hereSIC95N for more information about indexfiles.
